AbstractReportGenerator Structure

All generators extend AbstractReportGenerator which provides:

  • HTML template generation with CSS/JS
  • Report context with JAR content access
  • Statistics grid and section writing utilities
  • Consistent report structure and styling

Enhanced Features

Advanced generators support:

  • Security Analysis: Cross-reference with security data
  • Scheduling Analysis: Integration with @Scheduled analysis
  • Messaging Analysis: JMS/Kafka/RabbitMQ integration
  • Async Analysis: @Async and thread safety analysis
